Description
===========

Acid branded Trac theme using the TracThemeEngine plugin

Settings
========

Set up trac.ini with the following::

    [components]
    acidtheme.* = enabled

    [theme]
    theme = acid

    [header_logo]
    alt = acidwiki
    height = -1
    link = http://yoursite.com/
    src = theme/acid_logo.png
    width = -1

    [project]
    icon = theme/favicon.ico
