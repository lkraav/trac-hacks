Emoticons Support for Trac
==========================

This plugin provides support for graphical emoticons (aka smilies) in any Wiki
text in Trac.

To install this plugin, run `setup.py bdist_egg` and copy the egg file in the
`dist` directory into the `plugins` directory of the Trac environment. See
http://projects.edgewall.com/trac/wiki/TracPlugins for more options for
installing Trac plugins.

The icons provided by this plugin were originally created for the IRC client
Colloquy (http://colloquy.info), and are used here under the terms of the GPL.
