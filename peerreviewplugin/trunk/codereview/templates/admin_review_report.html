Review Report

Settings for Review Reports

Document Variables

You may use the following variable in title and subject which will be replaced with the actual value when creating a document.

$$reviewid: ID of the review

$$review_name: review name

Example: Report for Review $$reviewid

Document Template

The template path is not valid. A default template will be used instead: $template_default

The path must be readable by Trac. If no template is specified a default will be used.

Setting for File Lists

Document Template

The template path is not valid. A default template will be used instead: $filelist_template_default

The path must be readable by Trac. If no template is specified a default will be used.
