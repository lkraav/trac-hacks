QATracker is a Trac 0.11 plug-in used to manage QA Test Cases
via the native Trac interface and workflow.

This plugin assumes the following:
1. Trac 0.11
2. Genshi >= 0.5
3. Python >= 2.5
4. setuptools are installed and PATH'd properly
