Wiki Import - Configuration

Select instance to import pages from

You can import pages either from a preconfigured instance or specify the path to any existing instance.

Configuration

More instances can be added in trac.ini. Syntax documentation is available at : http://trac-hacks.org/wiki/WikiImportPlugin#Configuration.

You can define preconfigured instances by adding them in your project's trac.ini. Syntax documentation is available at : http://trac-hacks.org/wiki/WikiImportPlugin#Configuration.


Environment must reside on the same filesystem and be readable by whoever runs the web server.
